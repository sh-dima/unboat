# Changelog

## 0.1.1

* Fixed entities such as [chicken jockeys](https://www.youtube.com/shorts/RjKGhWY9VUo) not existing - angered entities will only leave boats and minecarts.
* Fixed the plugin not working if bStats could not be initialized.
* Added support for Minecraft version 1.21.9.

## 0.1.0

* Mobs inside boats will leave the boat when aggravated by another entity.
* Mobs that are already aggravated will not enter boats.
